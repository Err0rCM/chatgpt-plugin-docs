# 目录结构

chatgpt-plugin 默认生成的文件树如下所示 :

```
chatgpt-plugin
├── apps
├── config
├── node_modules
├── prompts
├── resources
│   ├── content
│   │   ├── Bing
│   │   ├── ChatGPT
│   │   └── static
│   │       ├── css
│   │       ├── fonts
│   │       ├── js
│   │       └── picture
│   ├── conversation
│   ├── help
│   └── img
└── utils
    ├── emoj
```
