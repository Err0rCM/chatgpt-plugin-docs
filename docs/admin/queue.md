# 队列操作

有时候遇到某个问题卡住了或者报错无法找到对话之类的可以查看或移出队列，也可以查看现在有哪些人正在和机器人对话



## 查看对话列表

::: tip 提示

目前API3模式下支持[切换对话](/admin/chat)

:::

方法：`#chatgpt查看对话列表`

发送后机器人会回复目前的对话列表及对话id



## 移出队列

::: tip 警告

仅API3模式下有效

:::

方法：`#移出chat队列首位`

发送后当前正在思考的问题会被移除并跳转到下一个



## 清空队列

::: tip 警告

仅API3模式下有效

:::

方法：`#移出chat队列首位`

建议卡死的时候使用，可以直接清空全部等待队列
