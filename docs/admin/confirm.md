# 问题确认

因为chatgpt一次只能回复一个问题，有些情况下比如网络不畅，服务器卡顿等会导致机器人等回复缓慢，我们无法知道是由于网络问题还是机器人问题，因此问题确认可以帮助我们了解回复队列情况



## 开启问题确认

设置方法：`#chatgpt开启问题确认`

开启后，机器人会以回复模式告知正在思考的问题，如果有多个问题正在等待，会提示前方队列还有多少问题

设置方法：`#chatgpt关闭问题确认`
