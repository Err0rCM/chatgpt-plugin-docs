# 问题确认

问题确认功能可以帮助我们确认机器人的回复状态，以便更好地了解回复队列情况和检查是否存在网络问题或机器人问题。

## 开启问题确认

您可以通过发送指令 `#chatgpt开启问题确认` 来开启问题确认功能。

开启后，机器人将以回复消息的方式告知正在思考的问题，并会在回复消息中提供有关前方队列中等待回复的问题数。

## 关闭问题确认

您可以通过发送指令 `#chatgpt关闭问题确认` 来关闭问题确认功能。
